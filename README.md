# Mida API Docs (Docusaurus)

This docs site is built with [Docusaurus](https://docusaurus.io/docs) and is focused on Mida Public V2 API endpoints.

## Install

```bash
npm install
```

## Local development

```bash
npm start
```

Then open `http://localhost:3000/docs/intro`.

## Build static docs

```bash
npm run build
```

Build output will be generated in `build/`.
